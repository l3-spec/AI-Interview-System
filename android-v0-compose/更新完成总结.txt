🔧 技术改进

1. Hero区域结构优化
   之前: Column + statusBarsPadding
   现在: Box + Column (渐变效果更好)

2. 卡片间距控制
   之前: Arrangement.spacedBy(5.dp)
   现在: Spacer(5.dp) (精确控制)

3. 组件命名优化
   之前: CreatePostDock
   现在: CreatePostButton (更语义化)

4. Padding设置优化
   • 标题区域: horizontal 4dp
   • 作者区域: start 4dp, end 4dp, bottom 4dp
   • 底部留白: 140dp (为导航栏预留)
